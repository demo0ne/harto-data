# harto-data

Data and assets for [Harto](https://github.com/demo0ne/harto) — card-based to-do for Heartopia.

- **cards/data.json** — card definitions
- **assets/images/** — card images (by pack: Daily, Daily-NPC, Weekly, etc.)
- **main.js**, **main.css** — app logic and styles

Loaded via jsDelivr: `https://cdn.jsdelivr.net/gh/demo0ne/harto-data@main/`
